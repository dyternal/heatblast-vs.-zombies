# Heatblast vs. Zombies

A 2D platformer game made in Unity 3D.

## Controls

- `W` or `Up Arrow` to jump
- `A` or `Left Arrow` to move left
- `D` or `Right Arrow` to move right
- `Space` to shoot
- `Esc` to pause
- `R` to restart
- `Q` to quit
